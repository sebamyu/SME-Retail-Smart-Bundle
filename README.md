# 🛒 Smart Bundle & Cross-Sell Recommendation Engine
**Data Science / AI Solution Proposal for Retail SME**

โปรเจกต์นี้เป็นการออกแบบ Data Science Solution เพื่อแก้ปัญหาตะกร้าสินค้าขนาดเล็ก (Low Basket Size) ในธุรกิจค้าปลีก โดยใช้เทคนิค **Market Basket Analysis (Apriori Algorithm)** ในการค้นหาความสัมพันธ์ของสินค้าที่มักจะถูกซื้อคู่กัน เพื่อนำไปต่อยอดในการทำโปรโมชั่น (Product Bundling) และเพิ่มยอดขายเฉลี่ยต่อบิล (Maximize Revenue: AOV)

## 📁 Files in this Repository (ไฟล์ในโปรเจกต์นี้)
1. **`SME_Retail_CrossSell.ipynb`** : โค้ด Python ฉบับเต็ม (Google Colab) ภายในประกอบด้วย
   - Data Dictionary
   - Mock Data Generation (Pattern Injection)
   - Exploratory Data Analysis (EDA) & Data Quality Check
   - Market Basket Analysis Model (Apriori)
   - Mock Dashboard Sketch
2. **`Slide_Proposal.pdf`** : สไลด์นำเสนอแผนธุรกิจและแนวทางการแก้ปัญหา

## 🚀 How to Run (วิธีดูผลลัพธ์)
สามารถกดคลิกที่ไฟล์ `.ipynb` ในหน้านี้เพื่อดูโค้ดและผลลัพธ์แบบสถิต (Static) ได้ทันที หรือหากต้องการรันโค้ดใหม่ ให้เปิดไฟล์ `.ipynb` แล้วคลิกที่ปุ่ม **"Open in Colab"** ที่อยู่ด้านบนสุดของไฟล์ เพื่อรันสภาพแวดล้อมบน Google Colab
